## Overview

Affinity Designer Advanced is a professional vector and raster design application that handles illustration, UI design, icon creation, and brand asset production in a single native application with the precision and capability that professional design work demands.

The vector persona provides a complete path-based drawing environment with full Bézier curve control, boolean operations, and the geometric precision that logo design and icon work requires without the performance overhead of web-based alternatives.

Pixel persona switches to a raster editing environment within the same document, allowing raster texture, photo manipulation, and pixel-level detail work to be performed alongside vector elements without exporting to a separate application.

Artboard and symbol management supports complex multi-screen and multi-format design projects, with master symbols that propagate changes consistently across every instance throughout the document.

For graphic designers, UI designers, and illustrators who want the full capability of professional vector design tools without subscription pricing, Affinity Designer Advanced is the design application that matches professional workflow requirements at a one-time cost.

## Performance

Affinity Designer balances performance with system resource usage intelligently. Operations are handled in the background, keeping your Mac responsive for everything else. Efficient memory management and smart caching reduce overhead during extended work sessions.

Apple Silicon acceleration is fully utilized on M-series Macs, delivering noticeably faster performance. On Intel machines, hardware acceleration is used wherever available.
